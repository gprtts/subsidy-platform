# Subsidy Platform

Платформа для сбора, хранения и предоставления информации о государственных субсидиях.

## Возможности
- Парсинг субсидий (пример: Москва)
- REST API только для чтения
- Telegram-бот (UI)
- CLI-утилита (демо)
- PostgreSQL 16

## Требования
- Python 3.12
- PostgreSQL 16

## Установка
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
